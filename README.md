# Dotfiles

## Setup

> NOTE: Before running the installation script (`install_mac.sh` or `install_linux.sh`), make sure to set the `DOTFILES` variable at the top of the script to your desired dotfiles directory path.

For macOS:

```bash
chmod +x install_mac.sh
./install_mac.sh
```

For WSL/Debian/Ubuntu:

```bash
chmod +x install_linux.sh
./install_linux.sh
```
